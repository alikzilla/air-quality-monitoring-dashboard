# Air Quality Monitoring Dashboard

Next.js + TypeScript + SCSS dashboard with terminal style UI, shadcn-style components, and three themes (light, dark, system).

## Pages

- Main: charts and combined metrics (AQI, temperature, humidity, CO2, CO)
- Sensors: sensor table with statuses and values
- Logs: simulated real-time stream
- Events: historical event timeline
- Settings: toggles/switches for runtime preferences

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
